## ✨ Overview

**Wellness Medicals** is a high-fidelity, web-based Hospital Management System built with a "VitalGlass" aesthetic. It prioritizes user experience, aesthetic excellence, and operational efficiency across multiple hospital roles.

The system provides a unified interface for doctors, nurses, pharmacists, and administrators, ensuring seamless data flow and clinical accuracy.

---

## 🚀 Key Modules & Features

### 👤 Role-Based Portals
- **🛡️ Admin Portal**: Comprehensive control over personnel, departments, and system logs.
- **🩺 Doctor Portal**: Clinical workspace for consultations, patient history, and prescription management.
- **👥 Staff Portal**: Efficient handling of patient registrations and frontline operations.

### 📅 Clinical Management
- **Appointments**: Interactive scheduling system with real-time status tracking.
- **Patient Registry**: Centralized patient records with detailed medical history.
- **Consultation Workspace**: High-fidelity interface for doctors with integrated pharmacy and lab links.

### 💊 Operational Tools
- **Pharmacy & Inventory**: Stock management with autocomplete search and stock level alerts.
- **Lab & Diagnostics**: Request and track lab reports with professional print-ready layouts.
- **Billing & Invoices**: Automated billing system for consultations and medication.

---

## 🎨 Design Philosophy: "VitalGlass"

Wellness Medicals uses a modern **Glassmorphic** design language characterized by:
- **Translucent Surfaces**: Soft background blurs and high-contrast text.
- **Ambient Lighting**: Dynamic background orbs and subtle gradients.
- **Micro-interactions**: Smooth transitions and hover effects for a premium feel.
- **Responsive Layouts**: Optimized for both high-resolution clinical monitors and mobile devices.

## 🛠️ Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/arshinmrelju/HMS.git
   ```
2. Open `index.html` in your browser.
